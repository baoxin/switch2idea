# 更新日志

本文档记录 "Switch2IDEA" 扩展的所有重要更新。

## [1.0.0] - 2024-03-20

### 新增功能

- 快速在 IDEA 中打开当前文件
- 自动定位到 VS Code 相同的光标位置
- 右键菜单直接打开文件到 IDEA
- 一键打开整个项目到 IDEA

### 配置选项

- IDEA 程序路径设置
  - Windows 默认: `C:\Program Files\JetBrains\IntelliJ IDEA\bin\idea64.exe`
  - macOS 和 Linux 默认: `idea` 命令
- 快捷键设置
  - 打开文件: `alt+shift+o`
  - 打开项目: `alt+shift+p`
